# maven-pom-sorter README

Simple extension that sorts the top-level XML elements for a **pom.xml** according to the [Apache Maven convention](https://maven.apache.org/developers/conventions/code.html#POM_Code_Convention).

## Features

Sorts **pom.xml** files on save, or on execution of `maven-pom-sorter: Sort pom.xml` in the Command Palette.
